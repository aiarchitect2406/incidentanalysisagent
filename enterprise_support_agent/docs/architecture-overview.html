Architecture Overview — Enterprise Support Agent on GEAP

Coordinator + sub-agents pattern. Four GEAP pillars composed end-to-end. Modeled on Google's published multiagent-ai-system reference architecture.

Build

ADK · Skill Registry · MCP

Scale

Agent Runtime · Sessions (short-term)

Govern

Agent Identity · Agent Gateway · Agent Registry · Model Armor (network layer)

Optimize

Cloud Trace · Cloud Logging · Observability · Topology · Eval
